基本参数设置

autMan后台地址：http://autMan的域名或ip:autMan端口/fanli/static

1.京东转链设置(可指令设置，可后台web设置)

  1. set fanli jd_union_id xxxx (说明：京东联盟ID)
  2. set fanli jd_union_appkey xxxxx(说明：京东联盟appkey)
  3. set fanli jd_union_secretkey xxxxx(说明：京东联盟secretkey)

2.淘宝转链设置(可指令设置，可后台web设置)

  1. set fanli zhe_appkey xxxxx（说明：折淘客appkey）
  2. set fanli zhe_sid xxxxx(说明：折淘客sid)
  3. set fanli tb_union_pid xxxxxxx(说明：淘宝联盟下的推广位，必须是选择的淘宝授权帐号下的PID，格式为：mm_56655336_24530010_191526926)

3.拼多多转链设置(可指令设置，可后台web设置)

  1. set fanli client_id XXXX (说明：多多进宝client_id)
  2. set fanli client_secret XXXX (说明：多多进宝client_secret)
  3. set fanli pid XXX (说明：多多进宝pid)

重要提示：以上参数设置完成后要重启、重启、重启（重要的事说3遍）才能生效，否则使用的是开箱初始化的参数
