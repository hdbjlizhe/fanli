转链参数

京东/淘宝转链

  1. set fanli jingpinku_appid xxxxx (说明：京品库appid)
  2. set fanli jingpinku_appkey xxxxx (说明：京品库appkey)
  3. set fanli jd_union_id xxxx (说明：京东联盟ID)
  4. set fanli jd_union_key xxxxx (说明：京东联盟API授权key)
  5. set fanli jd_union_appkey xxxxx(说明：京东联盟appkey)
  6. set fanli jd_union_secretkey xxxxx(说明：京东联盟secretkey)
  7. set fanli apikey xxxxx (说明：订单侠apikey)
  8. set fanli zhe_appkey xxxxx（说明：折淘客appkey）
  9. set fanli zhe_sid xxxxx(说明：折淘客sid)
  10. set fanli tb_union_pid xxxxxxx(说明：淘宝联盟下的推广位，必须是选择的淘宝授权帐号下的PID，格式为：mm_56655336_24530010_191526926)

拼多多

  1. set fanli client_id XXXX (说明：多多进宝client_id)
  2. set fanli client_secret XXXX (说明：多多进宝client_secret)
  3. set fanli pid XXX (说明：多多进宝pid)

重要提示：以上参数设置完成后要重启、重启、重启（重要的事说3遍）才能生效，否则使用的是开箱初始化的参数
