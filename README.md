# 介绍
autMan是【AI大脑】和【机器人】双轮驱动的可扩展自进化的个人助理解决方案。
- **个人 AI 助理**：通过本地运行并连接常用通讯软件（如 QQ、WX、Telegram）来提供服务，其核心能力在于高度个性化、自适应以及‌自我进化‌。
- **自动化工作流‌**：通过“技能”（Skill）功能，可以编排多步骤的任务流程，例如自动清理邮件、预订机票、监控市场动态并进行自动化交易等。‌
- **跨平台交互与智能响应‌**：支持通过qq、微信、微信客服、公众号、钉钉、飞书、tg客户端、tg机器人、Discord、Slack、Kook、Poe、Messenger、Synology Chat Bot、Pagemaid-Pyro等应用（含自己通过编写适配器接入的其他社交媒体）进行语音或文字交互，并能智能选择回复方式（如语音请求用语音回复）。支持关键词处理回复和事件处理操作，支持SSE连接，实时监听所有对接IM的消息。具备日志实时显示功能，方便调试。
- **自我进化与功能扩展‌**：用户可直接要求其添加新功能，它能自主完成autMan插件开发，例如配置模型支持、安装技能模块。 ‌扩展插件支持html、ECMAScript5、nodejs、typescript、python3、php、shell、golang等，不仅限这些编程语言，内置了插件市场，即装即用。支持自定义路由路径的微服务，即自定义路由，并自行编写处理逻辑。
- **自动化任务**：内置脚本运行容器，支持python3、javascript、shell、typescript 等多种语言脚本的定时任务管理，支持虚拟多容器，同时支持单脚本不同变量的并发和同变量分组并发，也支持对接青龙面板并进行管理。自动执行日程管理、邮件推送、RSS 源监控等复杂流程。 ‌
- **多模型支持与集成‌**：兼容openai、ollama、Claude、Gemini 等主流大模型，并可通过 MCP 服务器集成外部服务，如控制智能家居设备或访问邮件账户。 ‌
- **内置全可视化代理池**：维护可用代理，支持代理源自动休眠。支持隧道代理服务和API获取代理IP服务。
- **内置返利应用**：支持京东、淘宝、拼多多、外卖等返利（查询当前价格，历史价格，平台比价）支持二维码扫码购买，收益查询。
